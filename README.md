# 😶‍🌫️ CalmariaSPA

Projeto desenvolvido durante o curso de **CSS** da [Alura](https://www.alura.com.br/).  
A proposta é um SPA online fictício com navegação simples e intuitiva — incluindo botões animados e seção de contato.
